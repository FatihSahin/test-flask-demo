You can generate unit tests using TestFlask.CLI. 

Example command for Demo.QuickPay.Tests => QuickTests_Auto.cs is as follows

> testflask unit -a http://localhost:12079 -p QuickPayDemo -c QuickTests -n Demo.QuickPay.Tests -m jit

Use testflask --help to get help about options. 

Go to https://testflask.github.io/test-flask-cli for more information.
